# IconResizer
This provides the way to resize your app icon.

## environment
python 3.6.1  
pip pillow

## Install ##
```
$ git clone https://github.com/arabian9ts/IconResizer
$ chmod 755 IconResizer/resizer
```

## Use ##
example...
```
$ resizer IMG_NAME 16 32 48
```

### Try sample ###
after install...
```
$ cd IconResizer/sample
$ ./resizer sample.png 10 20 30
```
