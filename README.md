# IconResizer
This provides the way to resize your app icon.

## environment
python 3.6.1  
pip pillow

## Install ##
```
$ git clone https://github.com/arabian9ts/IconResizer
```

## Use ##
example...
```
$ python resizer.py IMG_NAME 16 32 48
```

### Try sample ###
```
$ git clone https://github.com/arabian9ts/IconResizer
$ cd IconResizer/sample
$ python ../resizer.py sample.png 10 20 30
```
